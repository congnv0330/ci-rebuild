# CI-Rebuild
Custom Codeigniter Framework

Package using:
- [Guzzle, PHP HTTP client](https://github.com/guzzle/guzzle)
- [Blade Template](https://github.com/jenssegers/blade)

## Required
- composer

## Start
- develop
```
composer install
```
- production
```
composer install --optimize-autoloader
```
## Using Laravel-mix
- [Doc](https://laravel.com/docs/6.x/mix)
```
npm install
```
## Using ciscript
```
php ciscript help
```
